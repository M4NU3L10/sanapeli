❓ Miten lisään uusia sanoja?

Voit luoda paketin helposti esim. tekoälyllä antamalla sille nämä ohjeet. Claude voi luoda tiedoston suoraan, muut antavat tekstin, jonka laitat muistioon, ja tallentaessa laitat nimen loppuun ".json". Sen jälkeen sinulla on kaksi vaihtoehtoa:

Tapa 1 – Tuo paketti -painike
Klikkaa ⬇️ Tuo paketti ja valitse .json-tiedosto koneeltasi. Paketti kopioituu automaattisesti oikeaan kansioon.

Tapa 2 – Kopioi itse kansioon
Klikkaa 📂 Avaa kansio. Kopioi .json-tiedostosi avautuvaan kansioon, sitten käynnistä ohjelma uudelleen.

Paketin rakenne (JSON)

{
  "name": "Paketti nimi",
  "sourceLanguage": "Suomi",
  "targetLanguage": "Saksa",
  "sourceFlagEmoji": "🇫🇮",
  "targetFlagEmoji": "🇩🇪",
  "words": [
    { "source": "talo", "target": "das Haus", "article": "das", "word": "Haus" },
    { "source": "mielellään", "target": "gern", "article": null, "word": "gern" },
  ]
}
