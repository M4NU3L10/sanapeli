Käyttöohjeet

Näin käytät Sanastopeliä

1. Asennus

Asenna suoraan tältä sivustolta

Asennus tapahtuu painamalla etusivun Lataa Sanastopeli -painiketta.

1

Avaa etusivu

Siirry takaisin etusivulle ja paina Lataa Sanastopeli.

2

Käynnistä asennusohjelma

Avaa ladattu Sanastopeli Setup 1.0.0.exe ja seuraa asennusvaiheet.

3

Käynnistä sovellus

Asennuksen jälkeen avaa Sanastopeli normaalisti työpöydältä tai Käynnistä-valikosta.

Vaatimus: Windows 10 tai uudempi (x64).

2. Käytön aloitus

Pelaaminen käyntiin

Valitse sanapaketti aloitusnäkymästä.
Valitse pelimuoto alavalikosta (kääntökortit, rakennuspeli, monivalinta tai kirjoitus).
Paina Aloita ja vastaa sanoihin pelimuodon mukaan.
Edistyminen tallentuu automaattisesti jokaiselle paketille.

3. Uusien sanojen lisääminen

Lisää omia sanapaketteja (.json)

Sovelluksen oman ohjeen mukaan voit lisätä paketin kahdella tavalla:

Tapa 1 – ⬇️ Tuo paketti: Paina sovelluksessa Tuo paketti ja valitse JSON-tiedosto. Paketti kopioituu automaattisesti oikeaan kansioon.
Tapa 2 – 📂 Avaa kansio: Paina Avaa kansio, kopioi JSON-tiedosto avautuvaan kansioon ja käynnistä sovellus uudelleen.

Paketin JSON-rakenne:

{
  "name": "Paketti nimi",
  "sourceLanguage": "Suomi",
  "targetLanguage": "Saksa",
  "sourceFlagEmoji": "🇫🇮",
  "targetFlagEmoji": "🇩🇪",
  "words": [
    { "source": "talo", "target": "das Haus", "article": "das", "word": "Haus" },
    { "source": "mielellään", "target": "gern", "article": null, "word": "gern" }
  ]
}

4. Pelimuodot

Valitse itselle sopiva tapa harjoitella

✏️ Kirjoitus: Kirjoita käännös itse (artikkeli voidaan hyväksyä joustavasti).
🔡 Monivalinta: Valitse oikea vastaus vaihtoehdoista.
🪧 Kääntökortit: Käännä kortti ja arvioi oma osaaminen (← ja → toimivat pikanäppäiminä).
🏗️ Rakennuspeli: Rakenna sana kirjainpaloista oikeaan muotoon.
