Käyttöohjeet

Näin käytät Sanastopeliä: kääntökortit, sanaharjoitus ja pelimuodot

1. Asennus

Asenna sanaharjoittelusovellus suoraan tältä sivustolta

Kääntökortti- ja sanaharjoitussovelluksen asennus tapahtuu painamalla etusivun Lataa Sanastopeli -painiketta.

1

Avaa etusivu

Siirry takaisin etusivulle ja paina Lataa Sanastopeli.

2

Käynnistä asennusohjelma

Avaa ladattu Sanastopeli Setup 1.0.0.exe ja seuraa asennusvaiheet.

3

Käynnistä sovellus

Asennuksen jälkeen avaa Sanastopeli normaalisti työpöydältä tai Käynnistä-valikosta.

Vaatimus: Windows 10 tai uudempi (x64).

2. Käytön aloitus

Pelaaminen käunteohjeet ja pelimuotojen valinta

Valitse sanapaketti alkunäkymästä — valittavissa eri kieliparit, esim. saksa, ranska, jne.
Valitse pelimuoto alavalikosta: kääntökortit, rakennuspeli, monivalinta tai kirjoitusharjoitus.
Paina Aloita ja vastaa sanoihin valitsemallasi pelimuodolla. Jokainen pelimuoto toimii eri tavalla.
Edistyminen tallentuu automaattisesti jokaiselle sanapakettille, joten voit jatkaa harjoittelua milloin tahansa.

3. Uusien sanojen lisääminen

Lisää omia sanapaketteja ja sanaharjoittelumateriaalia (.json)

Sanastopeli-sovellukseen voit lisätä omia JSON-sanapakettekansiaan. Sovelluksen ohjeen mukaan sanaharjoittelu tapahtuu kahdella tavalla:

Tapa 1 – ⬋️ Tuo paketti: Paina sovelluksessa Tuo paketti ja valitse JSON-tiedosto. Sanapaketit kopioituvat automaattisesti oikeaan kansioon.
Tapa 2 – 📂 Avaa kansio: Paina Avaa kansio, kopioi JSON-sanapaketit avautuvaan kansioon ja käynnistä sovellus uudelleen.

Sanapaketin JSON-rakenne ja muoto:

{
  "name": "Paketti nimi",
  "sourceLanguage": "Suomi",
  "targetLanguage": "Saksa",
  "sourceFlagEmoji": "🇫🇮",
  "targetFlagEmoji": "🇩🇪",
  "words": [
    { "source": "talo", "target": "das Haus", "article": "das", "word": "Haus" },
    { "source": "mielellään", "target": "gern", "article": null, "word": "gern" }
  ]
}

4. Pelimuodot

Kaikki pelimuodot: kääntökortit, monivalinta, kirjoitus ja rakennuspeli

✏️ Kirjoitusharjoitus: Kirjoita käännös itse — tehokas tapa oppia kirjoitusta ja sanastoa. Artikkeli voidaan hyväksyä joustavasti.
🔡 Monivalinta-harjoitus: Valitse oikea vastaus vaihtoehdoista — nopea ja tehokas sanaharjoitus.
🪧 Kääntökortit: Käännä kortti ja arvioi oma osaaminen — klassinen kertaamisen muoto (vasen ja oikea nuoli toimivat pikanuolikkeina).
🏗️ Rakennuspeli: Rakenna sana kirjainpaloista oikeaan muotoon — haastava ja hauska sanaharjoitus.
